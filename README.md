# Employee Management System (EMS)

A modern, responsive Employee Management System built using React, Redux, and Tailwind CSS.  
The application provides an intuitive dashboard, employee management features, and profile-based navigation.

---

## Project Overview

The Employee Management System (EMS) is a single-page web application designed to manage employee information efficiently.

It allows an admin user to:
- View workforce insights on a dashboard
- Manage employees (add, edit, delete)
- Filter and search employees
- Control employee active/inactive status
- Access profile actions such as Settings and Logout

The project focuses on clean architecture, predictable state management, and a responsive UI.

---

## Tech Stack Used

### Frontend
- **React (v19)** – Component-based UI
- **Redux + React Redux** – Global state management
- **React Router DOM** – Client-side routing
- **Tailwind CSS (v4)** – Utility-first styling with Vite integration
- **Lucide React** – Icon library

### Tooling & Build
- **Vite** – Development server and bundler
- **ESLint** – Code quality and linting
- **PostCSS + Autoprefixer** – CSS processing

### Data & Utilities
- **Axios** – HTTP requests (mock data)
- **LocalStorage** – Demo authentication state
